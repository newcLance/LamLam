- 章节栏第零章亮、后续锁；翻阅江湖入口置灰；此间录三项仅「铺卷一览」开放。

## 【关键教训】三大 GBK 编码翻车点

Windows GBK 控制台跑 Python 遇到 emoji/utf-8 特殊字符会 UnicodeEncodeError：
- **fetch-bili-videos.py**：视频标题 emoji（🤡）崩。已在脚本顶部加 `sys.stdout.reconfigure(encoding="utf-8", errors="replace")` 永久修好。
- **verify_asset_links.py**：❌⚠✅ emoji 崩。运行前 `export PYTHONIOENCODING=utf-8` 绕过。
- **规律**：pipeline 类脚本前一律 `export PYTHONIOENCODING=utf-8`。

外部权威源 `game-platform-analysis/screenshots/` 被沙箱标记为 Blocked。已在项目内新建 `screenshots_local/`，build.py 增加 5b 步 `shutil.copytree(local_ss, DIST/screenshots, dirs_exist_ok=True)` 合并覆盖。**今后新游戏封面双写位置**：`screenshots_local/{slug}/cover.webp` + `dist/screenshots/{slug}/cover.webp`。

- HTML 也可能返空（服务端反爬轮流），换关键词再试。

识别 IDENTIFY / 1.1 背景

背景 · 从枪战到怪谈

17 年的 CF 射击 IP 一脚踹进 UE5 次世代恐怖片，世界观核心是「虹蚀」物质——它腐蚀城市、腐蚀理智，也腐蚀玩家对那张老地图的所有熟悉感。

1 · 游戏世界观架构

近未来 · 虹蚀污染 · 夜幕山庄

故事发生在近未来的当代世界——这里没有外星人、没有末日核战，唯一的不正常是一种叫「虹蚀」的物质突然在世界各地爆开。被它沾上的城市会慢慢腐烂、长出触手；被它沾上的人会一边发疯一边掉理智。游戏舞台是被封锁的夜幕山庄，前一秒还是哥特式废弃别墅，下一秒月色一变，墙缝里就钻出克苏鲁式的肉变怪物。
近未来虹蚀污染夜幕山庄克苏鲁微恐
2 · IP 剧情

不是兵，是被推进恐怖区的平民

"这不是一个关于 CF 老情怀的故事，是一个关于「连那张你打了十年的地图都开始陌生」的故事。"
你扮演的不是穿制式军装的特种兵，而是一群被迫进区的平民——可能是个厨师、记者、或者派出所警察。手里没有什么先进装备，燃烧瓶得自己拿瓶子加破布做，消音器要去废墟里翻零件。一边躲怪物一边躲其他人类小队，队友也可能因为理智崩溃突然举枪打你——三方混战，活着出去就是赢。

3 · 角色介绍

三方对抗的角色版图

人

调查员小队（玩家）

厨师、记者、派出所警察等平民身份；穿户外便装而非制式军装；自制武器为主。

使命：进入虹蚀区搜刮 → 拼出真相 → 活着撤离。

怪

玩家操控的怪物

非对称玩法的另一端——克系触手、肉质变异、人形残骸混合而成的"过去 CF 地图上根本不存在"的存在。

使命：阻止人类撤离。

兵

另一支调查员小队

需要的资源、撤离点、剧情线索——和你的小队完全重叠。三方混战，PvPvE 由此成立。

使命：先撤的赢。

⚠ 第四种威胁：你的队友

理智值跌到底，队友会在压力到极限的瞬间举枪向你——这是 CF：虹 把"友军"也变成叙事危险源的核心设计。

定位 POSITION / 2.1 世界观高概念

世界观高概念 · LOOK · HOOK · BOOK

用一句话告诉所有人——这游戏好看在哪、好玩在哪、好理解在哪。

LOOK
好看 / 视觉钩子

UE5 Lumen 下哥特废弃别墅 × 克系触手肉变——纯黑底色 + 金属石青暗调 + 唯一的猩红 Logo 血雾笔触。

抓眼标识：红色"虹"字 + 调查员手电光柱 + 触手剪影。

HOOK
好玩 / 玩法钩子

不是又一个 FPS——是枪不够用、理智才是真血条的微恐射击。

核心循环：进区 → 搜刮 → 推剧情 → 撤离，理智值跨局影响。

BOOK
好理解 / 故事钩子

"老兵也得在月色里护好自己的脑子。"

17 年老 CF 地图被「虹蚀」侵蚀，平民被推进封锁区——熟悉地图 + 不熟悉恐怖 + 熟悉枪。

定位 POSITION / 2.2 世界观美学定义

世界观美学 · 克苏鲁微恐 × CF 次世代

把东欧哥特废墟和洛夫克拉夫特式触手肉变硬塞进一座 17 年的射击 IP 里——这是《虹》的视觉一句话定义。

角色美学

民间调查员 × 异变怪物

人类一侧是厨师/记者/警察等平民，户外生存/城市便装风格，去军事化是核心规则。怪物一侧融合克系触手、肉质变异、人形残骸。奥摩等经典角色 UE5 重塑。
物件美学

自制武器 × 拾荒装备

燃烧瓶 = 玻璃瓶 + 破布，消音器去废墟翻零件。装备和角色外观会被搜刮影响，强调求生而非装备先进。CF 经典枪械作为工具感而非军事感。
场景美学

哥特废墟 × 克系入侵

夜幕山庄哥特尖顶 + 长廊 + 舞会大厅；墙缝渗黑色脉络、吊灯下挂肉芽、月色翻红时整栋楼像活物呼吸。Lumen 全局光照、沥青褐 + 凝血红，红色 = 危险标识。

"审美趋势"共识是如何发展来的

克苏鲁视觉本不是射击品类的常客——下面这条线说明它怎么一步步走进战术射击。
1928
洛夫克拉夫特《克苏鲁的呼唤》
确立"不可名状之物 / 触手 / 理智崩溃"三件套。
1992
《鬼屋魔影》
早期 3D 恐怖游戏，克系美学进入电子游戏品类。
2015
《血源诅咒》
证明克系 + 硬核动作可以融合——"哥特 + 克系"的视觉教科书。
2017
《逃离塔可夫》
撤离循环 + 物资博弈成型，非对称多方对抗的紧张氛围范式建立。
2018
《湮灭》《遗传厄运》
克系生态变异与家庭恐怖 × 仪式邪教的电影级视觉范本。
2023
《心灵杀手 2》
把哥特恐怖与现代叙事推到 3A 量级——暗夜小镇恐怖的当代标杆。
2026
《CF：虹》
克苏鲁 × 战术射击 × 老 IP 重塑——UE5 Lumen 落成"老 IP 转型 + 微恐射击新品类"。

定位 POSITION / 2.3 定位风格

定位风格 · 某某主义 + 象限定位

从设计原则、关键词雷达图、风格情绪版到象限位置——把"虹"按死在视觉光谱里那一个特定位置。

遵循的"某某主义"

暗调写实恐怖主义 · Dark-Realism Horror

介于《心灵杀手 2》的暖中性暗调与《逃离塔可夫》的写实搜刮之间——既写实又超自然，既怀旧又陌生。

① 写实做底

材质 / 光照 / 动态 严格遵循 UE5 写实管线，不进入卡通或风格化区间。

② 微恐做调味

恐怖只在"关键时刻"压一下——靠光、声、变异肉芽，不靠 jumpscare。

③ CF 做骨

枪械手感、经典角色、关卡空间感保留 CF 基因，不为风格牺牲射击爽快。

构建 BUILD / 3.1 UI 设计思路

UI 设计思路 · 极简 HUD × 不可靠叙事者

UI 不只是显示数据，它是恐怖体验的延伸——理智值低，UI 就开始撒谎。

风格定义

极简 · 写实 · 不可靠

仅保留血量 / 弹药 / 理智值三项核心数据。理智值用大脑图标呈现，下降时屏幕边缘渗血色噪点 + 准星抖动。背包系统是格子式搜刮界面。低 San 值时 UI 本身会扭曲、闪烁。
对价值观的继承

CF 的工程感 + 实用主义

沿用 CF 端游"枪是工具"的工程化美学——金属边、磨砂质感、白底黑字的弹药计数。背包不浪漫，像翻一个真实的旧帆布包。
加入主观表达

UI 也会"发疯"

理智低，UI 开始撒谎：弹药数字闪一下变成奇怪符号、地图上多出不存在的撤离点、队友头像变成红色剪影。UI 不再可信——这是《虹》UI 最锋利的设计点。

怪物视角

切换到热感 / 异色视野——红绿对比，几何块状描边人类位置；与人类界面形成鲜明分裂。

构建 BUILD / 3.2 配色方案 · v1 真实采样

配色方案 · 基于官方 UI 像素提炼

本节覆盖了早期"暗蓝绿+猩红"的初稿假设。色板提炼自 cfh.qq.com 先导预约页三屏 UI 的像素级量化采样——全部色值都来自真实素材，不再凭直觉脑补。

A · 提炼素材

三张官方先导预约页截屏 + 各自的色带

HOME-1 · 雾林伫立调查员剪影于湿冷松林——湿绿+土褐底，胸甲深石青，Logo 血红是唯一高饱和点
HOME-2 · 暗角特写大量纯黑（93%）+ 一道月光绿灰高光，暗调写实标杆
HOME-3 · 篝火室内97% 纯黑 + 左侧橙红篝火光——唯一一处明确"暖光源"，给整体冷调色谱加了一道生人锚

关键修正 · 与早期"暗蓝绿+猩红"假设的差距

① 实际底色不是冷蓝绿，而是偏暖中性的"沥青褐 / 烟黑 / 金属石青"三色循环——更接近《Alan Wake 2》《Resident Evil 7》，而非《血源》冷青。
② 纯黑占据 75%–97%——官方策略是"黑场为主，主体浮岛"，所有 UI 信息都浮在黑里。
③ 红色不是亮红，而是干涸血红 #390909–#721D20——Logo 血雾笔触是唯一允许的高饱和符号。
④ 隐藏的橙红篝火光（#FF6A2C 系）是色谱里唯一暖光源——"人还在这里"的生命锚点。
⑤ 月光绿灰 #B4C9C1 在三屏里都出现一抹——它是 Logo 红的"安全色对位"。

7% · 自然辅色
70% 黑场——所有 UI 浮于黑底，建立"凝视感"
22% 中性暗调——主体 / 装备 / 卡片底
7% 自然辅色——月光、肤色、苔藓、可读文本
≤1% 危险红——Logo / 倒计时 / 关键警告，绝不滥用
D · 应用规则 · DO 与 DON'T

所有素材保持同一调性的"硬条款"

DO · 应当

红只用一次，且只在视觉锚点

  • 每张 KV / 海报上，红色面积不超过 5%，且唯一锚点是 Logo「虹」或关键警告
  • UI 中红色仅用于：低血量 / 倒计时 / 死亡 / 严重警告
  • 红色出现时必有黑场围拢（不接其他高饱和色）
DO · 应当

黑场为主，主体浮岛

  • 所有信息卡片 / 角色立绘 / 视频缩略图先在黑场上"打洞"，再放主体
  • 主体周围保留4–8% 的纯黑呼吸空间，绝不到边
  • 背景大面积暗角（vignette）是必选——半径不小于画面 60%
DON'T · 不得

不要冷蓝 / 紫 / 霓虹绿

  • 禁用赛博朋克霓虹（青蓝紫）——会破坏写实暗调氛围
  • 禁用高饱和绿色——只允许 #B4C9C1 月光绿灰这种低饱和
  • 禁用纯白 #FFFFFF 大面积——文字白以 #E6D9D8 替代
DON'T · 不得

不要让红色叠红色

  • Logo 红 + 警戒红同框 = 视觉打架——必须用黑场或月光绿灰隔开
  • 红色文字不许压在红色背景上——可读性归零
  • 篝火橙不得与血红毗邻——用枪钢灰过渡

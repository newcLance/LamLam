Overview

同一款产品，三个分析视角

这三份报告不是互相替代的方案选型，而是对《彩虹六号：攻势》国服首测从不同维度切入的分析。宏观视角回答"10 年老 IP 国服化的市场机会与品牌策略"；首测引流回答"B站/抖音/微博/小红书等外围平台 90 天怎么把人引到 WeGame"；主战场阵地回答"WeGame 商店页 + 游戏专区 + 自研启动器三个核心阵地怎么设计"。三份报告拼在一起才是完整拼图。

推荐阅读顺序

如果时间有限，建议按从战略到执行的顺序阅读：先看「宏观视角」建立全局认知（特别是国服化为什么现在 + 反作弊作为差异化武器），再看「主战场阵地」理解 WeGame 和启动器的核心转化路径，最后看「首测引流」补全外围拉人节奏。如果你只关心某个具体岗位的执行细节，三份报告都是自包含的，可以直接跳读对应视角。
